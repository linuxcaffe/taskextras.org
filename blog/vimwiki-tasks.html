Vimwiki-tasks is a taskwarrior accessory that extends task-listing to vim and vimwiki. The project page is here: https://github.com/teranex/vimwiki-tasks

This is a test of the ':InsertTasks < filter >' command, that when used within a vimwiki file will generate a task-list of < filter >. Note that this feature is presently almost undocumented.
